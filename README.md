# Tic-Tac-Toe Project

**Tic-Tac-Toe** is a typical game that everyone used to play at least once time in life.

## Installation

Use below command to install dependencies.

```bash
npm install
```

## Usage

Run Application on Dev Environment

```bash
npm run dev
```
